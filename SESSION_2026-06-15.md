## P0 — 완료

### 진행중 대전 해체 클랜 처리
- 상대 클랜이 해체된 대전 → 일반 방문자에게 자동 숨김
- 클랜장에게는 모든 진행중 대전 카드에 **취소/정리 버튼** 노출
- 카드 클릭 시 `/battle/[id]` 딥링크 (기존엔 `/battle` 목록으로만 이동)
- 대전 기록(완료)에서 해체된 상대 클랜명 → "해체된 클랜" 표시

### 클랜 탭 하단 스크롤 개선
- 탭 콘텐츠 영역 `minHeight: calc(100vh - 60px)` + 하단 패딩 160px
- 콘텐츠 적은 탭에서도 페이지가 충분히 길어져 스크롤 가능

### 클랜 공지 사라짐 버그 수정 (P0 추가 발견)
- **원인**: `clan_notices` 로드 시 `profiles(nickname)` PostgREST 조인이 FK 인식 실패 → 공지가 빈 배열 처리
- 작성 직후엔 state에 남아 보이다가 재방문하면 사라지는 증상
- **수정**: 조인 제거, 닉네임은 개별 조회로만 처리 + insert 에러 핸들링 추가

### 로그인 비밀번호 눈 버튼
- 비밀번호 input 우측에 눈/눈가림 토글 아이콘 추가

## 핵심 학습 / 주의사항

- **JSX 파일에 TypeScript 문법 금지**: `.jsx`에 `(c: any)` 쓰면 Turbopack 파싱 에러 → `.tsx`로 확장자 바꾸거나 타입 제거
- **PostgREST 조인은 FK 등록 필요**: `profiles(nickname)` 같은 조인이 FK 없으면 조용히 빈 배열 반환 → 개별 조회로 대체 권장
- **Vercel 환경변수 변경 후 반드시 Redeploy** 필요 (자동 배포 안 됨)
- **API 키는 절대 public 레포에 커밋 금지** → Vercel 환경변수로만 관리
- `is_hidden` 필터: `.eq("is_hidden", false)` 대신 클라이언트 `.filter(c => c.is_hidden !== true)` 사용 → 컬럼 없어도 안전
